Welcome to pqp’s documentation!¶

pqp (short for pourquoi pas?) provides tools for creating, visualizing, and performing inference on causal graphical models. The package aims to facilitate causal understanding for data scientists and researchers by providing an intuitive API built on top of performant and correct implementations of the underlying algorithms.
