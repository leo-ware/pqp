Graph¶

class pqp.graph.Graph(edges=[])¶

Bases: object

A causal graph

Example

>>> x = Variable("X")
>>> y = Variable("Y")
>>> g = Graph([
...     y <= x,
... ])
>>> g.idc([y], [x])
P(y | x)
Parameters:

edges (list of DirectedEdge or BidirectedEdge) – the edges in the graph
