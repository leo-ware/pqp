Graph¶

class pqp.graph.Graph(edges=[])¶

Bases: object

A causal graph

Example

>>> x = Variable("X")
>>> y = Variable("Y")
>>> g = Graph([
...     x <= y,
...     x & y,
... ])
>>> g.idc([x], [y])
