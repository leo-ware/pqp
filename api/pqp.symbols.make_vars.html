make_vars¶

pqp.symbols.make_vars(names)¶

Creates a list of variables from a list of names

Example

>>> make_vars(["x", "y", "z"])
[Variable("x"), Variable("y"), Variable("z")]
>>> x, y, z = make_vars("xyz")
[Variable("x"), Variable("y"), Variable("z")]
