pqp.data.domain.make_domain(domain_type, values=None)¶

Generates a domain object from a string and optional values

Options:

discrete: values must be specified continuous: min and max of values used to specify domain binary: values are ignored integer: min and max of values used to specify domain infer: attempt to guess

Parameters:

  • domain_type (str) – one of ‘discrete’, ‘continuous’, ‘integer’, ‘binary’ or ‘infer’

  • values (list) – the values of the domain
